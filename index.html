SGI STL 内存池分配器

参考 SGI STL 二级空间配置器思想，实现 MallocAlloc、PoolAlloc、SGIAllocator<T> 与简化版 MyVector，并在 Linux/g++ -O2 与 Windows/Visual Studio x64 Debug 环境下完成三轮十万级 push_back/pop_back benchmark。

Windows Debug 环境下自定义组合比 std::vector 快，但同容器对比里 SGIAllocator 慢于 std::allocator。因此项目结论应写成“双环境 benchmark 体现了平台、标准库实现和构建模式对性能的影响”，不要写成“内存池本身稳定快于标准分配器”。

Linux

cd ~/Desktop/src
g++ -std=c++17 -O2 -o benchmark main.cpp
./benchmark

Windows 数据来自 Visual Studio x64/Debug 运行结果。Debug 构建可能带来额外调试检查，因此不应和 Linux/g++ -O2 做绝对等价比较。
